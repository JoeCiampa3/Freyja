# cad/

Mechanical models for Freyja, developed in Fusion 360.

Models are exported to **STEP** for portability so they can be opened without proprietary software; native Fusion (`.f3d`) archives are included where useful for full feature history.

## Status

Current focus is the **pelvis assembly** — the first fabrication target and the structural anchor for the hip actuators analyzed in [`../docs/hip-joint-analysis.md`](../docs/hip-joint-analysis.md).

## Organization

One subdirectory per major assembly (e.g. `pelvis/`), each containing the STEP export and any supporting geometry. Lower-limb and spine assemblies will follow as they reach a stable state.
